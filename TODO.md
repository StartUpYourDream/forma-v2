# Forma V2 开发记录

## 核心概念澄清

```
用户 Votan
└── 创建团队 "JustDo"（我的创业团队）
    ├── 成员：Votan、Johnny、Agent们（PM/架构师/开发/测试）
    ├── 项目1: Forma（AI团队协作平台）
    │   ├── 聊天历史
    │   └── 代码文件
    └── 项目2: Tlist（待办事项应用）
        ├── 聊天历史
        └── 代码文件
```

**关键理解：**
- **Forma** = 项目名（不是团队名，不是产品名）
- **JustDo** = 团队名（创业团队/公司）
- 一个团队可以有多个项目
- 每个项目有自己的聊天 + 代码
